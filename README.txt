Pour faire fonctionner ce projet sur Render :
1. Assurez-vous d'ajouter les variables d'environnement suivantes :
   - OPENROUTER_API_KEY = votre clé OpenRouter

2. Dans le shell de Render, exécutez :
   npm install express body-parser cors dotenv openai pdfkit

3. Déployez avec app.js comme fichier principal.

⚠️ Remplacez le fichier index.html par votre version fonctionnelle une fois ce bug résolu.
